# Programing-Practices-2 🚀

A collection of C programming exercises focused on **Structures**, **Unions**, and **Nested Data Types**. This repository serves as a practical guide for mastering data organization and memory management in C.

---

## 📌 Features

This repository covers several key programming challenges, including:
* **Student Records**: Using arrays of structures to filter data by specific criteria (e.g., joining year).
* **Mathematical Operations**: Adding complex numbers by passing structures to functions.
* **Memory Management**: Demonstrating the fundamental differences between `struct` and `union`.
* **Logic & Comparison**: Comparing custom date structures (Day/Month/Year).
* **Data Hierarchy**: Implementing nested structures for complex real-world data modeling.
